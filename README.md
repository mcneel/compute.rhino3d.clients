This repository contains the project used to generate client libraries to communicate with [Rhino Compute](https://github.com/mcneel/compute.rhino3d). The project generates libraries for .NET, Python and JavaScript.

For more, see the [guides on the Rhino Developer Docs site](https://developer.rhino3d.com/guides/compute/).
